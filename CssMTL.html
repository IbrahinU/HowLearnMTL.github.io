Estilos en CSS


CSS (Cascading Style Sheets) es el lenguaje que se utiliza para aplicar estilos a una página web. Mientras que HTML define la estructura, CSS define la apariencia visual: colores, fuentes, márgenes, diseño, animaciones, etc.


Como se aplica CSS


  • En línea (inline): Se aplica directamente en una etiqueta HTML.
  • Interno (en el ): Dentro de una etiqueta style en el mismo archivo HTML.
  • Externo (archivo .css separado) Se enlaza un archivo .css al HTML.




Propiedades Comunes


Estas son algunas de las más usadas para dar estilo a los elementos HTML.

  • Color y Fondo: ("Color" y "background-color")
  • Tipografia: font-size:(Tamaño del texto); font-family:(Fuente del texto); font-weight: bold (Para ponerlo en negrita); text-align:(alineación)
  • Espaciado: margin: separa un elemento del exterior/ padding: separa el contenido del borde interior.
  • Bordes: "Border" y "Border-Radius"
  • Tamaño: Width:(Ancho) Height(Altura)
