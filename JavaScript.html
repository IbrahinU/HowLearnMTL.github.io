¿Qué es JavaScript?


JavaScript es un lenguaje de programación que permite agregar interactividad, dinamismo y lógica a los sitios web. Se ejecuta en el navegador del usuario y permite desde mostrar alertas hasta construir aplicaciones completas.

Eventos en JavaScript


Un evento es algo que ocurre en la página (como un clic, pasar el mouse, presionar una tecla, etc.).

CONSEJOS para ti


  • Siempre cierra las llaves {} y paréntesis () correctamente.
  • Usa la consola del navegador (F12 > Consola) para probar código.
  • Evita el uso excesivo de alert() en proyectos reales; es mejor modificar el DOM.
  • Prueba tu código poco a poco para encontrar errores más fácilmente.
