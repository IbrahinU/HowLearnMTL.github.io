Estructura básica de un documento HTML


HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear la estructura de las páginas web. Todo documento HTML sigue una estructura básica que permite a los navegadores entender y mostrar el contenido correctamente.


Ejemplo




Explicación de cada parte:


!DOCTYPE html: Declara el tipo de documento (HTML5).

html: Contenedor principal de toda la página.

head: Contiene información "oculta" como el título o los metadatos.

body: Contiene el contenido visible del sitio (textos, imágenes, etc.)



Etiquetas comunes de HTML


Las etiquetas HTML permiten estructurar y mostrar contenido. En este caso te voy a mostrar cuales son las mas usadas:




Descripción rapida de cada etiqueta


H1 a H6: Títulos de mayor a menor nivel.

P: Son para los párrafos.

Img: Son para las imágenes.

Ul / Li: Se utiliza para crea listas no ordenadas (con viñetas).

Ol: Son para listas numeradas.



De hecho, mira lo que veras acontinuación son listas ordenas y listas desornedas para que entiendes lo que te quiero decir

Etiquetas semánticas en HTML5


Las etiquetas semánticas describen el propósito del contenido, haciendo que el código sea más fácil de leer por humanos y máquinas (como Google o lectores de pantalla).




Ventajas


  • Mejora la accesibilidad.
  • Ayuda al SEO (posicionamiento en buscadores).
  • Hace el código más organizado y profesional.


Notaron como ahora si usé las viñetas para organizar el contenido, es para que vean que a medida que avanzen en su aprendizaje tambien notaran como la pagina avanza con ustedes.

Multimedia en HTML (video, audio, imágenes)


LHTML permite insertar contenido multimedia como imágenes, audio y video directamente en la página.




Consejos


  • Siempre incluye el atributo alt en imágenes para accesibilidad.
  • Usa controls para mostrar botones de reproducción.
  • Asegúrate de que los archivos estén bien vinculados y en formatos compatibles.
